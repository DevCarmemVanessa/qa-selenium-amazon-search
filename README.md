# 🤖 Automação de Busca na Amazon com Selenium

Projeto prático de automação de testes utilizando Selenium WebDriver com Python.

---

# 🎯 Objetivo

Automatizar o fluxo de pesquisa de produtos na Amazon validando o comportamento da aplicação.

---

# 🧪 Fluxo automatizado

✅ Abrir navegador Edge  
✅ Acessar Amazon Brasil  
✅ Localizar barra de pesquisa  
✅ Digitar produto automaticamente  
✅ Clicar no botão de busca  
✅ Validar título da página  
✅ Exibir resultado do teste

---

# 🛠️ Tecnologias utilizadas

- Python
- Selenium WebDriver
- Microsoft Edge
- VS Code

---

# 📚 Conceitos praticados

- Automação de testes
- Selenium
- Localização de elementos
- ID Locator
- WebDriverWait
- Expected Conditions
- Automação Web
- Validação automatizada

---

# 🚀 Estrutura do teste

1. Abrir navegador
2. Acessar Amazon
3. Encontrar elementos
4. Realizar pesquisa
5. Validar resultado
6. Finalizar automação

---

# 📌 Resultado esperado

O sistema deve realizar a pesquisa automaticamente e validar se a página carregou corretamente.
